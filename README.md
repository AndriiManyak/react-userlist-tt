# React-userlist-tt
- [DEMO](https://andriimanyak.github.io/react-userlist-tt/)

## Local deployment

### Dependencies
* Node v14.7.0
* NPM v6.14.7
* React v17.0.1
* React-dom": "^17.0.1",
* React-router-dom": "^5.2.0"
* Classnames v2.2.6
* Prop-types v15.7.2

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
